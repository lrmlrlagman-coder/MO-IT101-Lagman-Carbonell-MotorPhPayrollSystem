📋 Overview The MotorPH Payroll System is a Java application created to compute and present employee salary details using attendance records, employee data, and required government deductions. The program processes employee time logs, determines total working hours, calculates overtime and allowances, and applies the necessary deductions. Through this process, the system provides clear and complete payroll information for every employee.

✨ Features

👤 Employee Verification Confirms employee IDs by checking them against the company’s official records.

⏰ Time Tracking Records and processes attendance logs, including employee check-in and check-out times.

💰 Salary Calculation Calculates weekly pay based on the employee’s total hours worked and hourly rate.

⏱️ Overtime Management Computes overtime compensation using a premium rate of 125% of the regular hourly pay.

📑 Government Deductions Automatically calculates required government contributions, including:

SSS (Social Security System) PhilHealth Pag-IBIG Fund Withholding Tax
